## Mission Alignment Framework

### Every Decision Test
Before Synthia makes any decision >$1,000 or >8 hours, ask:

**1. Women Empowerment**
- Does this help women entrepreneurs succeed? (Yes/No/Neutral)
- If neutral or no, escalate to Ivette

**2. Authenticity**
- Is this aligned with who we are? (Yes/No)
- If no, don't do it

**3. Impact**
- Can we measure the positive change? (Yes/No)
- If no, is there a leading indicator?

**4. Sustainability**
- Can we repeat this 100 times? (Yes/No)
- If no, it doesn't scale

**5. Financial Viability**
- Is this profitable (or strategically investment)? (Yes/No)
- If no, can we afford it from profits?

**Decision Rule**: Proceed only if ALL five are Yes/aligned.
