## ARQUITECTURA DE CONTENIDO INSTAGRAM

### Los 4 Formatos que Mueven la Aguja

**Formato 1 — Reel de Demo (leads directos)**
```
[0-3s] Hook visual + audio trending
[3-20s] El problema o transformación
[20-35s] El proceso / solución en acción
[35-45s] Resultado con número o evidencia
Caption: Primera línea hook → CTA a bio / DM
```

**Formato 2 — Carrusel Educativo (saves + shares)**
```
Slide 1: Hook de portada (lo que verás)
Slides 2-8: Pasos, datos, o framework en porciones
Slide final: CTA + guardar para después
Caption: Contexto de 3 líneas + "Desliza →"
```

**Formato 3 — Testimonial / Caso de Éxito (conversión)**
```
Diseño: Quote del cliente con datos reales
Caption: Historia completa del before/after
Tag: Al cliente si acepta / con permiso
CTA: "¿Tú quieres resultados así? Bio"
```

**Formato 4 — Contenido de Valor / Tip (autoridad)**
```
Post estático o Reel corto con tip específico
Caption educativa con profundidad
Hashtags: Nicho + audiencia + ubicación
CTA: "Guarda esto. Lo vas a necesitar."
```

## ESTÉTICA DE MARCA KUPURI

**Paleta**:
- Negro profundo como base
- Magenta/fucsia para accentos clave
- Blanco limpio para texto
- Toques de cian para CTAs

**Tipografía**:
- Headers: Bold, limpio, legible en mobile
- Cuerpo: Legible, sin serif en digital
- Nunca Comic Sans, nunca gradientes rainbow

**Tono visual**:
- Premium pero accesible — no lujo inaccesible
- Real, no stock photos genéricas
- Detrás de cámaras auténtico de Ivette
- Resultados con captura, no con palabras
