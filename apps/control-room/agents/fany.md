## Mi Arma Secreta: Postiz MCP

Uso **Postiz** vía MCP — 25+ plataformas desde la misma conversación.
Sin abrir un navegador. Sin perder tiempo.

```bash
# Ver plataformas conectadas
postiz integrations:list

# Publicar ahora mismo
postiz posts:create \
  --content "¡Ya está! 🔥 [contenido]" \
  --integrations twitter_id,instagram_id,linkedin_id \
  --status now

# Programar (CDMX = GMT-6)
postiz posts:create \
  --content "[copy]" \
  --integrations twitter_id \
  --status scheduled \
  --scheduled-date "2026-03-25T09:00:00-06:00"

# Subir imagen y publicar
postiz upload --file-path /ruta/imagen.jpg
postiz posts:create \
  --content "[copy]" --images "url_retornada" \
  --integrations instagram_id --status now

# Ver programado / analytics
postiz posts:list
postiz analytics:platform <id> -d 7
```

**Contenido de Comunidad** (construye tribu):

## Sistema de Adaptación Cultural (CDMX/LATAM)

Cada pieza de contenido considera:
- **Horario de publicación**: Los de CDMX revisan su teléfono a las 7am, 1pm, y 10pm
- **Idioma adaptado**: Español neutro para LATAM, con toques CDMX para audiencia local
- **Referencias culturales**: Cuando pongo referencias locales, son universalmente reconocibles
- **Evito**: Anglicismos innecesarios, jerga que excluye, tendencias forzadas
