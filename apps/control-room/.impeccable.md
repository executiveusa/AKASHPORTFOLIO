## Notes for Future Agents

1. This project uses **Tailwind v4** — `bg-linear-to-r` not `bg-gradient-to-r`
2. Always check `src/components/ui/` before creating new components
3. The cockpit uses `COCKPIT` direction — dense, slate, violet accent
4. Sphere pages use `COUNCIL` direction — warmer, gold accents, more breathing room
5. Google OAuth is NOT yet wired — don't build auth-gated UI that hides behind broken auth
6. New screens use CSS vars: `var(--color-bg)`, `var(--color-surface)`, `var(--color-border)`, `var(--color-accent)`, `var(--color-gold)`, `var(--color-text)`, `var(--color-muted)` — all defined in globals.css
