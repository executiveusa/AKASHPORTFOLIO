## The Philosophy

> "We don't build websites. We build worlds."

Every visual decision must feel like it was made by a human with exceptional taste — not generated by an AI template engine. This standard applies to sphere rings, world backdrops, council UI, video generation prompts, and all design outputs Synthia creates for clients.

---

## The Aesthetic Target

**Inspirations (ranked):**
1. Linear — obsessive precision, dark surfaces, tight spacing
2. Raycast — tool-first, dense information, confident typography  
3. Vercel Dashboard — clean data, purposeful contrast
4. Pedro Pascal's wardrobe — understated, high quality, intentional
5. CDMX architecture (Kiosco Morisco, Candela shells) — structural beauty through form

**NOT inspirations:**
- SaaS template generators
- Generic dark-mode dashboards with neon gradients
- Crypto/web3 "holographic" UI
- AI art default output (purple voids, floating islands, lens flares)

## Typography

- **Primary font:** `Inter` or system UI sans
- **Heading size ladder:** 32 / 24 / 20 / 16 / 14px — no in-between
- **Body text:** 14px, `#a1a1aa` (zinc-400) on dark, line-height 1.6
- **Labels:** 11px, uppercase tracked, `#71717a` — only for metadata
- **No decorative copy** — if it explains the UI, delete it
- **Sphere names:** Always `font-medium`, sphere color only on the dot/ring — NOT on the text

---

## Spacing System

Use **4px base scale only:** 4 / 8 / 12 / 16 / 20 / 24 / 32 / 48 / 64px

Violations: random padding values (10px, 15px, 22px, 30px) → replace with nearest valid scale.

---

## Component Rules

### Sphere Rings
- Diameter: container-relative, max 64px in list, 96px on hover
- Glow: `box-shadow: 0 0 12px {sphere-color}40` — 25% opacity only
- Active state: ring width 2px solid `{sphere-color}`, inner glow only
- NO pulsing animations playing at rest — animate only on event

### Cards
- Border-radius: **8px max** (cards), never 16+
- Border: `1px solid #27272a` (zinc-800) only — no gradient borders
- Shadow: `0 1px 3px rgba(0,0,0,0.3)` — minimal, barely visible
- Background: `#18181b` (zinc-900) — one step above page bg

### Meeting World Backgrounds (Three.js / Blender)
- Fog density: 0.01–0.06 — enough to suggest depth, not enough to obscure
- Ambient light: 60% of key light intensity
- Ground: matte material, roughness 0.7–0.9
- Sky: gradient from `env.bgColor` to 15% lighter — NO texture maps in MVP
- Sphere agents: emissive material + 25% opacity point light of matching color

### Video Generation Prompts (Higgsfield / Kling)
- Always include: lighting quality, camera type, location anchor
- Always exclude: people, text overlays, watermarks, lens flares
- Quality prefix: "cinematic, 4K, architectural photography style"
- Never use: "futuristic", "neon", "cyberpunk", "glowing" (overused defaults)

## What Is Banned

| Pattern | Why banned |
|---------|------------|
| Glassmorphism (backdrop-blur + opacity) | AI default — instant recognition |
| Gradient backgrounds on cards | Visual noise |
| Purple void + floating spheres (default AI art) | Generic |
| Lens flares in video prompts | Overused |
| Uppercase + letter-spacing on navigation | Badge anti-pattern |
| KPI card grid as primary dashboard layout | SaaS template |
| Hero section inside admin panel | Context mismatch |
| Oversized border-radius (24px+) | Toy UI |
| Pill buttons everywhere | Neumorphic era vestige |
| Gold + black "luxury" clichés | Overused in LATAM SaaS |

## World Design Principles

1. **Architecture first** — structure creates beauty before materials do
2. **Scale correctly** — sphere agents are small relative to the world
3. **Light sources are intentional** — one key light, one fill, one ambient
4. **Atmosphere is density** — fog is subtle, not a veil
5. **Location DNA** — use the real location's color temperature (Kiosco: warm green, Zócalo: amber, Xochimilco: cool teal, Vallarta: Pacific gold)
6. **Motion is information** — animate only when something changes state
7. **GLB first** — all 3D exports as GLB for browser delivery via Three.js
